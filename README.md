# hakusai-RPC-Java
项目背景：RPC框架的目的是可以让客户端用服务端的方法就好像调用本地方法一样简单，服务端向注册 中心注册服务，消费端通过注册中心拿到服务相关信息，然后再通过网络请求调用服务端的服务。 
项目目标：开发一个RPC框架实现心跳检测，服务注册，远程方法调用，序列化和反序列化。 
项目技术：Netty+Nacos
